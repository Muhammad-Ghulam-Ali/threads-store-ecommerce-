# THREADS — Fashion Ecommerce Store

A modern, fully responsive ecommerce storefront built for a Pakistani casual wear brand.

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router DOM

## Features

- Responsive design (mobile sidebar, tablet, desktop)
- Product listing with filter by category and search
- Quick Add to cart with toast notification
- Cart sidebar with quantity management
- Product detail page
- Home page with hero, categories bento grid, and new arrivals

## Run Locally

git clone https://github.com/Muhammad-Ghulam-Ali/threads-store.git
cd threads-store
npm install
npm run dev
